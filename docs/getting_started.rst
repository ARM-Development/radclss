Getting Started
===============

Installation
------------

You can install radclss using pip:

.. code-block:: bash

   pip install radclss

Or install from source:

.. code-block:: bash

   git clone https://github.com/your-org/radclss.git
   cd radclss
   pip install -e .

Quick Start
-----------

Here's a simple example to get you started:

.. code-block:: python

   import radclss

   # Your example code here
   pass
